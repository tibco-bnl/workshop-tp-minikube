# Release Notes - v1.14.0

**Release Date:** February 27, 2026  
**TIBCO Platform Control Plane Version:** 1.14.0

## Overview

This release contains documentation and scripts aligned with **TIBCO Platform Control Plane version 1.14.0** for deploying TIBCO Platform on Minikube for local development and testing.

## What's Included

### Documentation
- Complete setup guides for Minikube cluster deployment with Control Plane and Data Plane
- Local development environment configuration
- PostgreSQL and MailDev setup for local testing
- Certificate management for local deployments
- Resource allocation and optimization guides

### Scripts
- Minikube cluster creation and configuration
- PostgreSQL and MailDev setup scripts
- Certificate generation and management
- Environment variable configurations
- Local DNS and ingress setup

## Key Features

- ✅ Full Minikube setup with TIBCO Platform CP + DP
- ✅ No cloud infrastructure required - runs completely local
- ✅ Perfect for learning, development, and testing
- ✅ Cost-effective development environment
- ✅ Workshop and training environment support
- ✅ Proof of concept demonstrations

## Compatibility

- **TIBCO Platform Control Plane:** v1.14.0
- **Minikube:** 1.30+
- **Kubernetes:** 1.24+
- **Helm:** 3.8+
- **Docker/Podman:** Latest stable version

## System Requirements

- **CPU:** 4+ cores recommended
- **Memory:** 16GB+ RAM recommended
- **Disk:** 50GB+ free space
- **OS:** macOS, Linux, or Windows with WSL2
